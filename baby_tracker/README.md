# 宝宝生活记录App - iOS安装说明

## 项目介绍

这是一款基于Web技术开发的宝宝生活记录应用，支持记录宝宝的日常活动（吃奶、喝水、睡眠、体温、洗澡等），并提供到点提醒和哄睡音乐功能。本应用设计为渐进式Web应用（PWA），可以直接安装到iOS设备的主屏幕上使用。

## 功能特性

- 📝 **记录功能**：记录吃奶、喝水、尿布更换、睡眠、体温测量、洗澡和用药等
- ⏰ **到点提醒**：设置定时提醒，如喂奶时间、服药时间等
- 🎵 **哄睡音乐**：内置多种舒缓音乐，帮助宝宝入睡
- 📊 **数据统计**：查看各项活动的统计数据和趋势
- 💾 **本地存储**：所有数据存储在本地，保护隐私安全
- 🌐 **离线使用**：安装后可在没有网络的情况下使用

## 将应用安装到iOS设备

### 方法一：直接从Safari浏览器安装（推荐）

1. **准备工作**：
   - 确保您的iOS设备已更新到最新版本
   - 确保应用已部署到可访问的Web服务器

2. **安装步骤**：
   - 使用Safari浏览器打开应用URL
   - 点击屏幕底部的**分享**按钮
   - 从分享菜单中选择**添加到主屏幕**选项
   - 为应用命名（例如："宝宝生活记录"）
   - 点击**添加**按钮
   - 应用图标将出现在您的主屏幕上

3. **首次使用**：
   - 点击主屏幕上的应用图标打开应用
   - 系统会提示请求通知权限，请点击**允许**以使用提醒功能
   - 设置您宝宝的基本信息（姓名、出生日期等）

### 方法二：使用TestFlight进行安装（开发测试版本）

如果您是开发人员或测试人员，可以使用TestFlight进行安装：

1. 将Web应用通过Xcode打包成iOS应用
2. 上传到App Store Connect并创建TestFlight版本
3. 向测试用户发送邀请链接
4. 用户安装TestFlight应用后，点击邀请链接接受并安装

## 部署说明

如果您需要自己部署此应用，请按照以下步骤操作：

1. **准备工作**：
   - 确保您的Web服务器支持HTTPS（iOS要求PWA必须通过HTTPS提供）
   - 确保服务器已配置正确的MIME类型支持

2. **部署文件**：
   - 将所有应用文件（html、css、js、images、sounds等）上传到您的Web服务器
   - 确保`manifest.json`和`service-worker.js`文件位于根目录

3. **服务端配置**：
   - 配置正确的缓存头（Cache-Control）以提高性能
   - 确保服务器不会阻止PWA相关文件的访问

## 应用权限说明

本应用需要以下权限才能正常工作：

1. **通知权限**：用于发送到点提醒
2. **音频播放权限**：用于播放哄睡音乐

## 数据备份与恢复

为防止数据丢失，建议定期备份应用数据：

1. **手动备份**：
   - 打开应用设置
   - 点击"导出数据"按钮
   - 将JSON备份文件保存到安全位置（如iCloud或其他云存储）

2. **数据恢复**：
   - 打开应用设置
   - 点击"导入数据"按钮
   - 选择之前备份的JSON文件
   - 确认导入操作

## 常见问题解答

### Q: 应用图标没有出现在主屏幕上？
**A**: 请确保您正确执行了添加到主屏幕的步骤，检查Safari浏览器版本是否支持PWA安装。

### Q: 提醒功能不工作？
**A**: 请检查是否已授予通知权限，确保您的设备没有开启勿扰模式。

### Q: 哄睡音乐无法播放？
**A**: 请确保已授予音频播放权限，并检查设备音量是否开启。

### Q: 安装后无法离线使用？
**A**: 首次使用时需要保持网络连接，确保应用完全加载并缓存。之后即可离线使用。

### Q: 如何卸载应用？
**A**: 在iOS主屏幕上，按住应用图标，选择"删除App"，然后确认删除。
